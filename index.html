This is a clearinghouse where

  • Charities can list their needs for Volunteers, and
  • Volunteers can sign up to help with event needs.

Event Administrators, do you need to fill certain roles for an event?

Volunteers, do you want a one-stop shop to identify needs and sign up to help?

This network is built for you.

Once approved as an administrator, you can create an event, give its description, assign it to a certain date, and then create roles and number of volunteers needed.

If you are a volunteer, you can review a calendar of all events open, look for roles that you are qualified to fill, and then sign up!

Because we use Facebook authentication, no need to register and duplicate info.
